# MultiTool-Grupo3
Proyecto Final de Git

Este proyecto tiene como finalidad mostrar el trabajo en equipo y el correcto uso de los repositorios remotos con los reporsitorios locales de cada integrante y una simulacion de como trabajan los desarrolladores hoy en dia y como pueden trabajar en equipo sin necesidad de estar juntos, destacando que Git es una herramienta versatil y eficaz.

El proyecto propone 5 mini-programas sencillos pero interesantes que se dividen de la siguiente manera:
ANALIZAR TEXTO,ADIVINA EL NUMERO (Victor del val LIDER)
CONVERSOR UNIDADES (Santiago Sarmiento README)
GESTOR NUMERO(Nacho Enguidanos dev )
GESTOR TAREAS"TEXTO"(Ivan garcia dev)


Que hace el proyecto?
Vamos a pasar por cada desarrollador para explicar como funciona y su importancia en el proyecto.
